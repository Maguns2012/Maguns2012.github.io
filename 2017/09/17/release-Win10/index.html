release Win10

发表于 2017-09-17 | 分类于 System |

  • 免费活动

    今天好友说有一个送皮肤的活动。可以自己去尝试刷一波免费的永久皮肤。正逢周末，闲来无事。打打游戏也是种消遣，打开电脑。才想起，前些天貌似无法连接wifi了，当时懒得理就没有收拾它。今天打开一看，哟，好好地。啥事没有。还是蛮开心的。

  • 艰难的游戏

    习惯的卡顿进入游戏，加上网速本身不给力。游戏玩的甚是恼火，打了一把我觉得我的胃液开始“翻腾了”。随即不在完了。

  • 大佬的Blog

    晚上看到大佬的微博更新了Ubuntu 16.04更新BBR的文章，不明觉厉。随逐在该本本的虚拟机上，开始搞起来。然后~Ubuntu起来的了。该电脑的菜单起不来了~。此刻我觉得我的胃液已经快到我的心口了。。。为了自己声明安全还是关机，放在一旁吃灰吧。

  • 归属

    深刻的觉得，此刻于此操作系统，此电脑不适合留在我的身边。想想还是整理一下，给老爸留着打打扑克好了。

  • 祝愿

    如果有来世，希望别再有这么恶心的操作系统。
